# Feature Detail: SelectDB Management

**Feature ID**: 006  
**Name**: SelectDB Management  
**Description**: Provide lifecycle management for Alibaba Cloud SelectDB instances and clusters.  
**Status**: Planned

## Overview

This feature aims to provide comprehensive Terraform support for Alibaba Cloud SelectDB, including managing instances, clusters, and other related resources. It involves updating existing resources and data sources to align with the latest API definitions in `cws-lib-go`.

## Latest Review

Initial creation of the feature.

## Key Changes

1. Update `alicloud_selectdb_instance` resource to support new API fields.
2. Update `alicloud_selectdb_cluster` resource to support new API fields.
3. Update `alicloud_selectdb_instances` data source.
4. Update `alicloud_selectdb_clusters` data source.

## Implementation Notes

- Must use `cws-lib-go` for API calls.
- Follow the provider's layered architecture (Resource -> Service -> API).
- Ensure backward compatibility where possible.

## Future Evolution Suggestions

- Support for more granular SelectDB resources if API allows (e.g., security groups, accounts).
