## Overview

该特征描述了项目使用的基于 Makefile 的构建系统，提供了一套完整的自动化工具链，包括代码格式化、跨平台构建、测试执行等功能，提升开发效率和代码质量。

## Key Changes

1. 实现跨平台自动构建（macOS、Linux、Windows）
2. 集成代码格式化和静态检查工具
3. 提供开发环境快速搭建能力
4. 支持单元测试和验收测试执行
5. 提供 Alpha 版本构建和发布能力

## Implementation Notes

- 支持自动检测操作系统类型并构建对应平台版本
- 集成 gofmt、goimports 等代码格式化工具
- 提供 dev、devlinux、devwin 等开发构建目标
- 支持测试执行和结果输出格式化
- 支持 Alpha 版本构建和上传到 OSS
