# Feature Detail: 强类型 API 调用

**Feature ID**: 005  
**Name**: 强类型 API 调用  
**Description**: 使用 cws-lib-go 库提供的强类型结构体进行 API 调用  
**Status**: Implemented  
**Spec Path**: .specify/specs/008-refactor-sls-resource-api/spec.md  
**Created**: 2025-12-02  
**Last Updated**: 2026-01-12

## Overview

该特征描述了项目中使用 cws-lib-go 库提供的强类型结构体进行 API 调用的设计和实现。通过使用强类型结构体替代弱类型的 map[string]interface{}，提高了代码的安全性和可维护性。

## Key Changes

1. 优先使用 cws-lib-go 提供的强类型结构体
2. 避免在新代码中使用 map[string]interface{} 作为主要载体
3. 在 Service 层封装 API 调用逻辑
4. 提供统一的错误处理机制
5. 支持类型安全的请求和响应处理

## Implementation Notes

- 所有新的 API 调用必须使用 cws-lib-go 提供的强类型结构体
- 禁止在新代码中使用 map[string]interface{} 或未约束的 interface{}
- 仅在对接遗留路径时可短暂使用弱类型，并需要在代码注释中说明理由
- 代码评审必须对新增的弱类型使用进行质疑与把关
